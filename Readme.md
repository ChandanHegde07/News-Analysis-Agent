# News Analysis Agent

An intelligent news analysis system built with **LangGraph** and **Google Gemini AI** that automatically gathers, analyzes, and generates comprehensive reports from multiple news sources.

## Features

- **Automated News Gathering**: Fetches articles from RSS feeds
- **Content Scraping & Cleaning**: Extracts and cleans article content using LLM
- **Trustworthiness Evaluation**: Assesses source credibility and fact-checking
- **Fact Extraction**: Identifies key information, entities, and events
- **Report Generation**: Creates structured analysis reports with insights

## Dependencies

- **langgraph**: State machine framework
- **langchain-google-genai**: Google Gemini integration
- **feedparser**: RSS feed parsing
- **newspaper3k**: Article scraping
- **beautifulsoup4**: HTML parsing
- **python-dotenv**: Environment management
